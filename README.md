# CUBE
CUBE is a free and simple timer for magic cubes

## Features
* Scramble Generator
    * 2x2x2
    * 3x3x3 (BLD, OH, FMC, Feet)
    * 4x4x4 (BLD)
    * 5x5x5 (BLD)
    * 6x6x6
    * 7x7x7
    * Rubik's Clock
    * Megaminx
    * Pyraminx
    * Square-1
* Statistics
    * Best time
    * Average of 5
    * Average of 12
